# Conflict Resolution

Staff should reduce conflict, not make it worse. Stay calm, factual, and professional.

## Conflict Standards

| Standard | Action |
| --- | --- |
| Stay Calm | Do not match player anger. |
| Stay Factual | Use evidence and rules. |
| Stay Neutral | Avoid assumptions. |
| Stay Private | Move sensitive issues to tickets. |
| Escalate | Ask higher staff before it gets worse. |

## Useful Phrases

```text
I am going to review the evidence before making a decision.
```

```text
This needs higher staff review, so I am escalating it.
```

> [!NOTE]
> Staff do not need to win arguments. Staff need to resolve situations correctly.
