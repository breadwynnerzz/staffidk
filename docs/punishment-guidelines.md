# Punishment Guidelines

Punishments must match the situation and stay consistent across all players. Do not punish from anger, bias, favoritism, or pressure.

## Punishment Matrix

| Situation | Action |
| --- | --- |
| Minor first issue | Reminder or warning. |
| Repeated minor issue | Formal warning. |
| Moderate violation | Warning, kick, or temp action. |
| Severe violation | Escalate for major punishment. |
| Staff disrespect | Handle calmly and document. |
| Exploiting or major toxicity | Immediate escalation. |

## Required Standard

- Use rules, evidence, and context.
- Document major punishments.
- Escalate if unsure.
- Keep similar cases consistent.

> [!DANGER]
> False, careless, or biased punishments may result in demotion or removal.
