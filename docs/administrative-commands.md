# Administrative Commands

Commands must only be used for real staff duties. Do not use commands for fun, intimidation, favoritism, or personal benefit.

## Command Rules

| Rule | Standard |
| --- | --- |
| Valid Reason | Every command needs a staff purpose. |
| Correct Target | Confirm the player first. |
| Proper Severity | Use only what the situation requires. |
| Logging | Document serious actions. |
| Approval | Get approval for restricted actions. |

## Examples

```text
/warn [player-id] [reason]
/kick [player-id] [reason]
/ban [player-id] [duration] [reason]
/bring [player-id]
/goto [player-id]
/freeze [player-id]
```

> [!DANGER]
> Command abuse may result in immediate suspension, demotion, or removal.
