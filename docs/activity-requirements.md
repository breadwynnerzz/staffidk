# Activity Requirements

Staff must stay active in Discord, server duties, and assigned responsibilities. Being online without helping does not count as activity.

## Counts As Activity

| Activity | Examples |
| --- | --- |
| Community | Welcoming players, answering questions, starting conversation. |
| Support | Handling tickets or directing players correctly. |
| Moderation | Reviewing reports and enforcing rules. |
| Internal Work | Training, meetings, evaluations, or documentation. |

## Inactivity

If you will be inactive, contact ownership or management before disappearing.

Include:

- Expected return date
- Short reason if needed
- Duties that need coverage

> [!WARNING]
> Repeated inactivity or poor communication can lead to warning, demotion, or removal.
