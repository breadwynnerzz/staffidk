# Warning System

Warnings correct behavior and create a record when issues continue. They should be professional, clear, and based on facts.

## Warning Types

| Type | Use |
| --- | --- |
| Verbal Warning | Minor first-time issue. |
| Formal Warning | Clear violation or repeated issue. |
| Final Warning | Serious pattern before removal. |

## Warning Notes

Include:

- Date and time
- Person involved
- Rule or expectation broken
- Evidence or context
- Staff member issuing it
- Follow-up needed

> [!WARNING]
> Warnings are accountability tools, not personal attacks.
