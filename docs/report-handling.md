# Report Handling

Reports must be handled with evidence, context, and server rules. Do not rely on opinions, friendships, or rumors.

## Report Process

| Step | Action |
| --- | --- |
| Receive | Confirm who and what is involved. |
| Collect | Ask for clips, screenshots, or logs. |
| Review | Check rules and context. |
| Decide | Apply action or escalate. |
| Document | Record the outcome if needed. |

## Evidence Rules

- Serious action needs clear proof.
- Unclear evidence should be escalated.
- Do not punish from incomplete context.
- Similar reports should have similar outcomes.

> [!WARNING]
> Major punishments must include evidence and reasoning.
