# Reusable Markdown Templates

Use these templates when adding future staff handbook pages.

## Standard Policy Page

```markdown
# Page Title

Brief professional summary of the policy or procedure.

> [!NOTE]
> Use this for important context that staff should remember.

## Expectations

| Expectation | Standard |
| --- | --- |
| Placeholder | Placeholder |

## Procedure

1. Placeholder step.
2. Placeholder step.
3. Placeholder step.
```

## Warning Notice

```markdown
> [!WARNING]
> Use this for policy reminders, approval requirements, and situations that require caution.
```

## Critical Notice

```markdown
> [!DANGER]
> Use this for termination-level issues, permission abuse, leaking information, or major staff violations.
```

## Command Block

````markdown
```text
/command [target] [reason]
```
````
