# Moderator Responsibilities

Moderators help maintain order, assist players, and enforce rules fairly. They should stay respectful, patient, and unbiased in every situation.

## Main Duties

| Duty | Action |
| --- | --- |
| Reports | Review evidence and apply rules. |
| Support | Help players with basic issues. |
| Rule Enforcement | Keep punishments consistent. |
| Escalation | Contact higher staff when needed. |
| Documentation | Log important actions if required. |

## Limits

Moderators should not issue major punishments unless policy allows it or higher staff approve it.

> [!WARNING]
> Trial Moderators must work under supervision and escalate serious situations.
